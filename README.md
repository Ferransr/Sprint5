# Sprint 5 | IT Academy - FrontEnd React

### INTRODUCTION

A coaching company is conducting an experiment in companies in Barcelona, measuring the impact of humour and fun on productivity.
They have asked us for a web application that shows help to employees before they start their working day.

### BUILD WITH

- Programación en Typescript y JS con ES6.
- Peticiones a APIs.
- Maquetación en CSS.
